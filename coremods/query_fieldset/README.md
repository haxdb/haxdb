mod_fieldset
===========

This module allows the creation of fieldsets.
We define a fieldset as  both a list of columns and a query.


Database
--------

Creates the following tables:
- **FIELDSET**
- **FIELDSET_COLS**


API
---

- /FIELDSET/list
- /FIELDSET/view
- /FIELDSET/save
- /FIELDSET/new
- /FIELDSET/delete
