mod_lists
===========

This module provides tables and api to create/edit/save LISTS.  LISTS can be used to provide valid values when storing data.


Database
--------

Creates the following tables:
- **LISTS**
- **LIST_ITEMS**


API
---

- /LISTS/list
- /LISTS/new
- /LISTS/save
- /LISTS/delete

- - - -

- /LIST_ITEMS/list
- /LIST_ITEMS/new
- /LIST_ITEMS/save
- /LIST_ITEMS/delete
